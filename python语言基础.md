# 变量

动态类型，不声明变量
Python 变量命名规则：

- 必须以字母或下划线字符开头，不能以数字开头
- 只能包含字母数字字符和下划线（A-z、0-9 和 _）
- 区分大小写

变量初次为其赋值时，创建变量

```python
x = 12
y = "python"

x1, y1 = 12, 13
x2, y2 = 14, "python"
x3 = y3 = z3 = "Orange"
```

## 位运算符

Bitwise Operators

- `&` 按位与
- `|` 按位或

- `^` 按位异或
- `>>` 按位右移 高位补最高位
- `<<` 按位左移 低位补零
